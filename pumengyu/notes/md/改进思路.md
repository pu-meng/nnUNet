# 小/极小肿瘤分割方法整理（2022–2026）

> 任务背景：nnUNet + LiTS 肝脏小肿瘤分割，外部验证 3D-IRCADb  
> 筛选标准：仅保留**直接针对小/极小目标**的方法，去掉泛用方法  
> 排序依据：改动成本低 × 对小目标收益大 × 与现有工作兼容性  
> 已放弃：BoundaryLoss（实验 fold_0/4 均低于 baseline）、TwoStage（Stage2 FP 爆炸且低于联合分割）

## 第一阶段：立刻可做（改动小，收益大）

### 1. nnUNet 过采样配置调整
**类别**：训练策略  
**论文**：nnUNet 原生支持，无需额外论文  
**原理**：提高小前景区域的 patch 采样概率，确保网络在训练中充分见到极小肿瘤。  
**操作**：
```python
# nnUNet dataset.json 或 trainer 中调整
oversample_foreground_percent = 0.5  # 默认 0.33，调高到 0.5~0.67
```
**现状**：`SmallTumorOversampleMixin`（thresh=6000 locs, 3x repeat）已在 TwoStage/BoundaryLoss trainer 中实现，但 **Dataset003 baseline 从未单独测试过此效果**，需补一个干净的对照实验。

---

### 2. Unified Focal Loss
**类别**：Loss 设计  
**论文**：*Unified Focal Loss for all-round improved detection and segmentation*，Medical Image Analysis，2023  
**原理**：统一框架下的 Focal Loss 变体，通过聚焦参数（focusing parameter）和不对称权重，自动平衡极小前景与大量背景体素之间的梯度贡献，无需手动调 class weight。  
**推荐原因**：不需要 per-sample 计算，只改 loss 函数，与任何 trainer 直接兼容，是 Size-stratified Loss 的更简单替代起点。  
**代码**：GitHub 有开源实现，PyPI 可直接安装。

---

### 3. Copy-Paste 小肿瘤数据增强
**类别**：数据增强  
**论文**：*Simple Copy-Paste is a Strong Data Augmentation for Instance Segmentation*，CVPR 2021；医学图像版本在 MICCAI 2023 有多篇专门针对 LiTS 的应用。  
**原理**：将极小肿瘤 ROI crop 出来，随机粘贴到同数据集其他正常肝脏区域，生成新的训练样本。  
**关键点**：
- 粘贴位置限制在肝脏 mask 内（避免生物学不合理）
- 粘贴时做简单的强度混合（泊松混合或简单加权）减少伪影
- 可将小肿瘤样本量增加 3–5 倍
- **在 LiTS 数据集上有论文验证有效**

**为什么最高优先级**：实验分析显示 13 个严重失败案例中 7 例为极小肿瘤（<5k 体素），3 例体素数 <1000，根本原因是训练样本中此类案例太少，数据层面的修复比 loss 调整更根本。

## 第二阶段：改动中等，收益显著

### 4. Size-stratified Loss
**类别**：Loss 设计  
**论文**：2025 arxiv 系列  
**原理**：按目标体积大小分层计算 loss 权重，体积越小的目标自动获得越高的惩罚系数：
```python
weight = 1.0 / (target_volume_voxels + epsilon)
```
**实现注意**：nnUNet 以 patch 为单位训练，需在 `train_step` 内对当前 patch 的 GT 动态估计肿瘤体积，比 per-scan 计算复杂，改动成本高于 Unified Focal Loss。  
**与 Unified Focal Loss 的关系**：可叠加，Focal Loss 处理 voxel 级不平衡，Size-stratified 处理 case 级不平衡。

---

### 5. U-Mamba（混合 CNN-SSM 架构）
**类别**：网络架构  
**论文**：*U-Mamba: Enhancing Long-range Dependency for Biomedical Image Segmentation*，arXiv 2401.04722，2024  
**作者**：Jun Ma, Bo Wang（Vector Institute，Toronto）  
**代码**：https://github.com/bowang-lab/U-Mamba  

**核心原理**：  
Mamba 基于状态空间模型（SSM），复杂度为 O(N)，相比 Transformer 的 O(N²) 可以在**高分辨率特征图**上建模全局依赖。

```
每个 U-Mamba Block：
  ResBlock（卷积，局部特征）
+ ResBlock（卷积，局部特征）
+ Mamba Block（全局长程依赖，线性复杂度）
```

**对小肿瘤的意义**：小肿瘤在 128³、64³ 的高分辨率特征图里还存在，Mamba 在此建模其与周围血管/边界的全局关系，而 Transformer 在这个分辨率下显存爆炸跑不起来。

**与 nnUNet 的兼容性**：U-Mamba 直接继承 nnUNet 自配置机制，代码结构高度兼容，替换 encoder block 即可。

**实践建议**：
```
第1层（最高分辨率）：只用 ResBlock（显存压力大）
第2–4层：U-Mamba block（分辨率适中，收益最大）
Bottleneck：U-Mamba block（全局上下文）
```

---

### 6. Large Kernel Attention (LKA)
**类别**：网络架构（轻量注意力）  
**论文**：*Visual Attention Network (VAN)*，CVPR 2023  
**原理**：将大卷积核（如 21×21）分解为三步：
```
深度可分离卷积 + 深度可分离空洞卷积 + 1×1 卷积
```
模拟大感受野 attention，但计算量远小于 self-attention。空洞卷积不做下采样，不损失分辨率，小肿瘤的局部纹理得以保留。  
**插入位置**：encoder 第 2–3 层（高分辨率层，小肿瘤还存在的地方）。

---

### 7. Overlap-aware Loss
**类别**：Loss 设计  
**论文**：MICCAI 2024  
**原理**：传统 Dice Loss 中，小目标区域容易被大目标或背景的预测"压制"，梯度贡献极小。Overlap-aware Loss 在计算时识别预测重叠区域，给被压制的小目标额外的梯度权重。

---

### 8. TTT-Seg（测试时训练）
**类别**：训练策略  
**论文**：*Test Time Training for Medical Image Segmentation*，2024  
**原理**：在推理阶段，利用自监督信号对模型做短暂更新（几个 gradient step），使模型适配当前测试样本的分布。  
**对你的意义**：LiTS（训练）→ 3D-IRCADb（外部验证）存在域偏移，TTT-Seg 可以显著改善跨数据集泛化，对小肿瘤的跨域检测尤其有帮助。

---

## 第三阶段：有余力时做（改动大，作为消融对比）

### 9. Deformable Attention (DAT)
**类别**：网络架构  
**论文**：*DAT: Deformable Attention Transformer*，CVPR 2022  
**原理**：attention 的采样点位置可学习，网络自动将注意力偏移到肿瘤边界等关键位置，而不是固定网格。3D 医学图像版本已有应用。

---

### 10. HD Loss 改进版（可微 Hausdorff Distance Loss）
**类别**：Loss 设计  
**论文**：HD Loss 系列，2023  
**原理**：原始 Hausdorff Distance 不可微，近似版本通过距离变换实现可微计算，专门惩罚边界偏差。小肿瘤边界占体积比例高，受益最大。

---

### 11. Tumor Synthesis（生成式数据增强）
**类别**：数据增强  
**论文**：MICCAI 2023–2024 多篇  
**原理**：用生成模型（GAN 或 Diffusion）在正常肝脏 CT 上合成虚假小肿瘤，显著扩充极小肿瘤训练样本。  
**难点**：需要额外训练生成模型，工程复杂度高。

---

### 12. SAM-Med3D 弱监督预标注
**类别**：数据  
**论文**：*SAM-Med3D*，2024  
**原理**：利用在大规模3D医学数据上预训练的 SAM-Med3D，对未标注的小肿瘤做自动预标注，再人工校验，扩充训练集。

---

### 13. Dynamic Sparse Convolution
**类别**：网络架构  
**论文**：基于 MinkowskiEngine 的医学应用，2024  
**原理**：只在有意义的体素（前景区域）上计算卷积，大量背景体素跳过计算。对小肿瘤的意义：梯度不被背景体素稀释，且可使用更高分辨率输入。

---

## Loss 组合建议（针对你的任务）

```
推荐组合（第一阶段）：
Total Loss = CE+Dice（nnUNet 默认）
           + λ₁ × Unified Focal Loss

推荐组合（第二阶段）：
Total Loss = CE+Dice（nnUNet 默认）
           + λ₁ × Unified Focal Loss
           + λ₂ × Size-stratified Weight
           + λ₃ × Overlap-aware Loss

参数建议起点：λ₁=0.5, λ₂=1.0, λ₃=0.3
```

## 实验路线建议

```
本周（1–2天，控制变量）：
  └── Dataset003 + nnUNetTrainer + SmallTumorOversampleMixin only → fold_0
      目标：单独确认过采样效果，分离已有实验的混淆变量

Week 1–2：
  ├── 加入 Unified Focal Loss → fold_0 对比
  └── 实现 Copy-Paste 增强 → fold_0 对比

Week 3–4：
  ├── 最优组合（过采样 + Copy-Paste + Focal Loss）→ 跑完整 5-fold
  └── 视结果决定是否加 Size-stratified Loss

Month 2：
  ├── 替换 encoder 为 U-Mamba（第2–4层）
  ├── 消融实验：U-Mamba vs baseline vs U-Mamba+Loss组合
  └── 外部验证 3D-IRCADb

Month 3（有余力）：
  ├── TTT-Seg 提升 3D-IRCADb 结果
  └── Deformable Attention 消融对比
```
