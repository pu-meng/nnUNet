# 项目进展与计划

> **任务**：LiTS（MSD Task03_Liver）肝脏小/极小肿瘤分割，nnUNet v2 3d_fullres，5-fold CV  
> **核心信号**：`separability_metrics.json`（wasserstein 等14个可分性指标，零泄露）驱动分层过采样  
> **硬约束**：官方测试集无标签 → 所有指标只靠 5-fold CV；难度信号绝不碰模型预测  
> **策略转向（本轮）**：CopyPaste 全面落后 Baseline，已放弃；改用分层过采样（StratifiedOversampleMixin）

## 二、已完成的改进（代码已实现）

### ✅ StratifiedOversampleMixin（本轮主线，推荐使用）
分层过采样，用 `separability_metrics.json` 中的 wasserstein 距离驱动重复次数：
- 无肿瘤 case（13个）：固定重复 4x（降低误报）
- 肿瘤 case：wasserstein 低（肿瘤难分）→ 重复多（最多 5x），wasserstein 高 → 重复少（最少 1x）
- 自然覆盖极小肿瘤 + 极大肿瘤 + 无肿瘤，无手设阈值

对应 Trainer：`Tr_SOS`、`Tr_SOS_UFL`

### ✅ 两总体可分性分析（本轮）
`separability_metrics.py` 计算 14 个指标（wasserstein, AUC, Bhattacharyya 等）× Dice 相关性：
- wasserstein: ρ=+0.291（最高），低 wasserstein = 肿瘤难分 = 难度高
- AUC: ρ=+0.265，overlap: ρ=-0.260
- 各指标统计上无显著差异（CI 半宽 ≈ 0.165 > 各指标间距）
  
### ✅ UnifiedFocalLossMixin（UFL）
CE+Dice 之上叠加 AsymmetricUnifiedFocalLoss，自动平衡体素级正负样本。  
参数：`UFL_LAMBDA=0.5, UFL_DELTA=0.5, UFL_GAMMA=0.2`（固定，未做消融）。

### ✅ DifficultyOversampleMixin（已被 SOS 取代，保留兼容旧 Trainer）
按 difficulty.json（overlap 排名）线性映射重复次数。  
**问题**：difficulty.json 不含无肿瘤 case；已被 StratifiedOversampleMixin 取代。

### ✅ CopyPasteMixin / DifficultyCopyPasteMixin（已放弃）
实验结果：CopyPaste(-2.8pp), CopyPaste_v2(-0.9pp), CopyPaste_Diff(-3.2pp)，全部低于 Baseline。  
**结论**：放弃 CopyPaste 方向，改用纯分层过采样。

### ✅ 新命名规范 `Tr_` 前缀（本轮）
结果目录从 `nnUNetTrainer_XXX__nnUNetPlans__3d_fullres` 缩短为 `Tr_XXX__nnUNetPlans__3d_fullres`。

## 四、待实现的改进（代码未写）

### ~~SmartCopyPaste~~（已放弃）
CopyPaste 系列（CopyPaste / CopyPaste_v2 / CopyPaste_Diff）全部低于 Baseline，
根本问题是粘贴引入分布偏移，不值得继续投入。  
改用纯分层过采样（StratifiedOversampleMixin）替代。

### 🔲 Size-stratified Loss（中优先级）
按肿瘤实例体积分配 loss 权重，小实例权重高。  
与 UFL 互补（UFL 处理体素级不平衡，Size-stratified 处理实例级不平衡）。  
实现需在 `train_step` 内做连通域分析，计算开销中等。  
**等 `Tr_DOS` 结果出来后再决定是否加。**

### 🔲 UFL λ 消融（低成本）
新建 `Tr_DOS_UFL_l03` / `Tr_DOS_UFL_l10`，固定 `UFL_LAMBDA=0.3 / 1.0`，  
对比当前 `UFL_LAMBDA=0.5`，确认最优 λ。

## 六、暂不做的方向（及原因）

| 方向 | 暂不做的原因 |
|------|------------|
| TTT-Seg（测试时训练） | 针对跨域泛化，LiTS 内部 CV 阶段无收益；推理时需反向传播，实现复杂 |
| 合并更多数据集 | LiTS 已是最大公开肝肿瘤数据集；其他数据集域偏移大，合并可能有害 |
| Kendall 可学习 λ | λ 是 Python float 不在计算图中，需改 optimizer + checkpoint，工程复杂度高，收益不确定 |
| Overlap-aware Loss | MICCAI 2024，无公开代码，实现细节不确定 |
| Diffusion Seg / Mask2Former | 推理慢 / 改动极大，不适合当前阶段 |
