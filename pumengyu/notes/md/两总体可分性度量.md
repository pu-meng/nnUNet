# 两总体可分性度量大全

> **问题**：把肿瘤体素和肝脏体素各看作一个总体，"这两个总体有多难区分"？
> **目标**：找出与分割难度（OOF Dice）相关性最强的指标，用来驱动 difficulty.json。
> **脚本**：`pumengyu/tools/data_analysis/separability_metrics.py`

---

## 一、问题形式化

```
总体 A：肿瘤体素 HU 值   x₁,...,xₙₜ  ~ P_tumor
总体 B：肝脏体素 HU 值   y₁,...,yₙₗ  ~ P_liver

可分性 ↑  →  模型容易区分  →  Dice 高
可分性 ↓  →  模型难以区分  →  Dice 低  →  难度高
```

---

## 二、参数化方法（假设正态分布）

### Cohen's d
$$d = \frac{\mu_t - \mu_l}{\sigma_{\text{pooled}}}, \quad \sigma_p = \sqrt{\frac{(n_t-1)\sigma_t^2 + (n_l-1)\sigma_l^2}{n_t+n_l-2}}$$

- d=0：等密度，最难；|d| 大：可分性高
- **问题**：假设两组方差相近（同方差）；CT 肿瘤分布常有双峰，不满足

### CNR（放射学标准）
$$\text{CNR} = \frac{|\mu_t - \mu_l|}{\sigma_{\text{liver}}}$$

- σ 取肝脏背景的标准差（噪声估计），是放射科最通用的可见性指标
- **问题**：仍只用均值，忽略肿瘤内部 HU 分布的复杂性

### d'（d-prime，信号检测论）
$$d' = z(\text{Hit Rate}) - z(\text{False Alarm Rate})$$

- 经典信号检测论（Signal Detection Theory）核心参数
- 正态等方差时等价于 Cohen's d
- **来源**：心理物理学，广泛用于医学影像 ROC 分析
- 与 AUC 的关系：$\text{AUC} = \Phi(d'/\sqrt{2})$（正态等方差假设下）

### Bhattacharyya 距离（正态闭合形式）
$$D_B = \frac{1}{8}(\mu_t-\mu_l)^2 \cdot \frac{2}{\sigma_t^2+\sigma_l^2} + \frac{1}{2}\ln\frac{\sigma_t^2+\sigma_l^2}{2\sigma_t\sigma_l}$$

- 两项分别度量"均值差"和"方差差"
- **优点**：不要求等方差；与 Bayes 误差率有指数关系（详见第五节）

## 三、非参数方法——分布距离（无正态假设）

### Histogram Overlap（你已有）
$$\text{overlap} = \sum_k \min(p_t[k],\, p_l[k])$$

- overlap=1：完全重合；overlap=0：完全不重合
- 等价于最优 Bayes 分类器的最大误判率

### Total Variation 距离
$$\text{TV} = 1 - \text{overlap} = \frac{1}{2}\sum_k |p_t[k] - p_l[k]|$$

- TV=0：相同分布；TV=1：完全不重叠
- overlap 的互补，"可分程度"的直接度量

### Bhattacharyya 系数 / 距离 / Hellinger
$$\text{BC} = \sum_k \sqrt{p_t[k] \cdot p_l[k]}, \quad D_B = -\ln(\text{BC}), \quad H = \sqrt{1 - \text{BC}}$$

- BC=1（DB=0，H=0）：完全重合；BC=0（DB=∞，H=1）：完全分离
- Hellinger 是真正的距离（满足三角不等式），BC 和 overlap 的平滑版本
- **优点**：对直方图稀疏 bin 比 min 更稳定（几何平均 vs 取最小值）

### Jensen-Shannon 散度（对称 KL）
$$\text{JS} = \frac{1}{2}\text{KL}(P_t \| M) + \frac{1}{2}\text{KL}(P_l \| M), \quad M = \frac{P_t + P_l}{2}$$
$$\text{KL}(P \| Q) = \sum_k p[k]\ln\frac{p[k]}{q[k]}$$

- JS ∈ [0, ln2]；JS=0：相同分布；JS=ln2：完全不重叠
- KL 散度不对称（KL(P‖Q)≠KL(Q‖P)），JS 是对称修复版
- √JS 是真正的度量（满足三角不等式）
- **信息论含义**：区分来自 P 还是 Q 时平均需要多少额外比特

### Kolmogorov-Smirnov 统计量
$$\text{KS} = \sup_x |\hat{F}_t(x) - \hat{F}_l(x)|$$

- 两个经验 CDF 的最大差距，范围 [0, 1]
- 对分布"最偏离的那一点"敏感，捕获局部差异
- 同时可做显著性检验（KS 检验 p 值）

### Wasserstein-1 距离（Earth Mover's Distance）
$$W_1 = \int_{-\infty}^{\infty} |F_t(x) - F_l(x)|\, dx$$

- 把一堆"土"从分布 P 移到分布 Q 的最小搬运代价
- 单位是 HU，物理意义直观
- **优势**：即使两分布完全不重叠也有意义（KL、JS 此时趋于无穷）
- 计算：`scipy.stats.wasserstein_distance(t, l)`，O(n log n)

## 四、非参数方法——判别力

### AUC（最推荐）
$$\text{AUC} = P(X_t > X_l) = \frac{U}{n_t \cdot n_l}$$
其中 $U$ 是 Mann-Whitney U 统计量

- AUC=0.5：随机猜，等密度，最难
- AUC=1.0：完全可分，最容易
- **折叠处理**：取 max(AUC, 1-AUC) 到 [0.5, 1]，消除低/高密度方向的差异
- **优点**：
  - 完全非参数，无分布假设
  - 直接等于"随机选一个肿瘤体素和一个肝脏体素，能猜对哪个是肿瘤的概率"
  - 与 Cohen's d 有解析关系：$\text{AUC} = \Phi(d/\sqrt{2})$（正态等方差时）
  - `sklearn.metrics.roc_auc_score` 或 `scipy.stats.mannwhitneyu` 直接算

### Bayes 误差率下界
$$\text{BayesErr} = \frac{1 - \text{TV}}{2} = \frac{\text{overlap}}{2}$$

- 任何分类器的理论**最低误判率**
- BayesErr=0.5：最难（随机猜）；BayesErr=0：完全可分
- 物理含义：即使用最优分类器，也至少有这个比例的体素会被误分

---

## 五、各方法与 Bayes 误差率的理论关系

| 方法 | 与 Bayes 误差率的关系 |
|------|----------------------|
| Overlap / TV | `BayesErr = overlap/2`，直接关系 |
| Bhattacharyya | `BayesErr ≤ √(PT·PL) = BC`，上界 |
| Hellinger | `BayesErr ≤ H²/2` |
| AUC | AUC=0.5 时 BayesErr=0.5；但不直接等价 |
| Cohen's d | 正态下：`BayesErr = Φ(-|d|/2)` |

## 六、分布形状特征

### 双峰系数（Bimodality Coefficient）
$$\text{BC} = \frac{\text{skewness}^2 + 1}{\text{kurtosis}_{\text{Pearson}}}$$

- 正态分布：BC = 1/3 ≈ 0.333
- 均匀分布：BC = 5/9 ≈ 0.556
- **BC > 5/9 提示双峰或重尾**
- 意义：肿瘤 HU 分布若双峰（坏死核+强化环），BC 高；等密度肿瘤分布窄，BC 接近正态

## 七、已有分析结果（per-case 汇总统计的天花板）

用 HU 均值、标准差等**汇总统计**计算的 4 个特征，与 Dice 的 Spearman ρ：

| 特征 | ρ | 95% CI |
|------|---|--------|
| contrast（均值差） | -0.318 | [-0.47, -0.15] |
| overlap（直方图重叠） | -0.295 | [-0.45, -0.12] |
| cohen's d | +0.288 | [+0.11, +0.45] |
| \|contrast\| | +0.284 | [+0.11, +0.44] |

**结论**：四者 |ρ| ≈ 0.28-0.32，差距 0.034 << CI 半宽 0.16，统计上无法区分。

**天花板原因**：per-case 汇总统计把体素分布压成一个数，信息大量丢失。

## 八、如何突破 0.3 天花板

```
现在（每 case 一个数）:
  overlap = 0.45   →  弱相关 ρ≈0.3

突破方向——用"分布级"特征:
  AUC（体素级判别力）         → 不压缩，直接度量可分概率
  Wasserstein（分布搬运代价）  → 捕获位置+形状信息
  双峰系数（形状）             → 检测坏死+强化环双峰结构
  KS 统计量（最大偏差点）       → 捕获局部差异
```

> 用数据决定哪个指标最强，再用它更新 `difficulty.json`。
