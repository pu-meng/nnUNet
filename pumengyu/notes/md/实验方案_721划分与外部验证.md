# 实验方案：7:1:2 固定划分 + IRCADb 外部验证

## 背景

放弃原有的 5-fold 交叉验证，改为固定比例划分，与主流医学影像论文保持一致。

---

## 数据集

| 数据集 | 用途 | 例数 | 标签类别 |
|---|---|---|---|
| MSD Task03 (Dataset003_Liver) | 训练 / 内部验证 | 131 | 0=背景, 1=肝脏, 2=肿瘤 |
| 3D-IRCADb | 外部验证 | 20 | 同上（已转 NIfTI） |

---

## 内部划分（7:1:2，分层抽样）

**随机种子 = 42，按有无肿瘤分层，确保三个子集比例一致。**

| 子集 | 比例 | n | 有肿瘤 | 无肿瘤 | 用途 |
|---|---|---|---|---|---|
| train | 0.7 | 92 | 83 | 9 | 模型训练 |
| val | 0.1 | 13 | 12 | 1 | nnUNet 训练监控 / early stopping |
| **test** | **0.2** | **26** | **23** | **3** | **论文报告 internal test set** |

> **注意**："2"对应的是 test（论文汇报），"1"对应的是 val（训练监控，不单独列在论文里）。

### 生成方法
```bash
python pumengyu/tools/make_split_721.py --dataset 3
```
- 输出：`preprocessed/Dataset003_Liver/splits_final.json`（fold 0，含 train+val）
- 输出：`preprocessed/Dataset003_Liver/split_info_712.json`（完整三子集记录，含 test case ID）

### 内部测试集推理
```bash
# 26 个 test case 从 imagesTr 直接推理

  -i /home/PuMengYu/nnUNet_workspace/raw/Dataset003_Liver/imagesTr \
  -o /home/PuMengYu/nnUNet_workspace/results_v2/test_preds/<Trainer> \
  -d 3 -c 3d_fullres -f 0 -tr <Trainer>
# 只对 split_info_712.json 里 test 列表的 case 计算指标
```

### 外部验证（IRCADb）推理
